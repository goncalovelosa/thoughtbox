# HDD Quick Reference

Compact command map for modular HDD.

## Main Command

```bash
/hdd <adr-number> "<title>" [--resume]
```

## Phase Commands

```bash
/hdd:research "<goal>"
/hdd:stage-adr "<title>"
/hdd:validate <staging-adr-path>
/hdd:decide <staging-adr-path>
```

## Module Paths

- `./modules/research-brief.md`
- `./modules/stage-adr-brief.md`
- `./modules/implement-brief.md`
- `./modules/validate-brief.md`
- `./modules/decide-brief.md`

## Required Checkpoints

1. Approve hypotheses after research.
2. Approve scope after staged ADR.
3. Confirm manual checks during validation.
4. Confirm accept/reject decision.
